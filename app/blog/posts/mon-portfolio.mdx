## Pourquoi ce portfolio ?

Tout simplement pour avoir une vitrine personnel, où je peux montrer mes projets, mes compétences et mes expériences pro mais aussi perso. C'est aussi un moyen de me challenger, pour apprendre de nouvelles choses et de m'améliorer.

## Le stack technique utilisé

J'utilise Next.js qui est un framework de React, ça permet de gérer le backend (côté serveur) et le frontend (le visuel) tout en un, le styling des pages est fait avec Tailwind CSS, ainsi que MDX pour l'écriture des pages du blog.

## Choix de Next.js

Bien qu'utiliser un framework comme Next.js pour un simple portfolio peut paraître exagéré, j'ai choisi de l'utiliser pour avoir rapidement une base pour mon portfolio, j'ai utilisé un modèle minimale de porfolio que je compte adapter à mes besoins. Il se trouve sur le site de Vercel (créateur de Next.js): [Next.js Blog Starter](https://vercel.com/templates/next.js/portfolio-starter-kit), malgré que ça reste basique, le référencement naturel (SEO) est déjà en place ce qui est avantageux pour la visibilité de mon portfolio sur les moteurs de recherche. Il y a également le flux RSS qui est déjà en place.
